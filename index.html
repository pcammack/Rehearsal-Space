REHEARSAL SPACE v5.3
WHAT'S NEW IN v5.3
· + Add piece: New piece name field at the bottom of the library. Type a name, press Enter or click + Add — file picker opens for audio immediately.
· JSON import fixed: Handles all backup formats including mr_lib_v4. Merges without overwriting existing pieces.
· Mini waveform player: Each recorded clip has a clickable waveform — click to seek, playhead tracks position. No more browser audio controls.
· Blend fixed: Blend toggle syncs reference to correct position. _downloadBlendClip and WAV export now present and working.
· Solo recording corrected: Recording from marker/segment no longer plays reference audibly — instrument only.
· Folder architecture, waveform tooltips, and marker annotations from v5.1 all retained.
WORKFLOW
1. Select a piece from the Library — the file picker opens automatically.
2. Press Play. Press M to drop markers while listening.
3. In Loops: pick Start → End markers, click Add. Click ⟳ to loop, ▶ to preview.
4. Add notes in the Notes panel — saved automatically.
5. Save via Library → + Save current.
MARKER NAMING
· Numeric: 1 → 2 → 3
· Alpha: A → B → C
· Dash-numeric: A-2 → A-3 → A-4 (subsections)
· Dash-alpha: 3-A → 3-B → 3-C (subsections)
· Placeholder: . (period) — excluded from cascade and loop dropdowns
