MUSIC REHEARSAL v4.1
WORKFLOW
1. Load audio file. Marker 1 placed at 0:00 automatically.
2. Press Play. Press M to drop markers while listening.
3. In the Segment Builder: pick Start → End from dropdowns, click Add. Or click "All adjacent" to pre-populate consecutive pairs.
4. Name segments (default: StartLabel→EndLabel). Check/uncheck to include in export.
5. Click Export Segments. Each checked segment becomes one MP3.
MARKER NAMING CONVENTION
Auto-increment rules when pressing M:
· Numeric: 1 → 2 → 3
· Alpha: A → B → C
· Dash-numeric: A-2 → A-3 → A-4 (suffix increments)
· Dash-alpha: 3-A → 3-B → 3-C (suffix increments)
· Placeholder: Use . (period) — excluded from cascade and dropdowns
· To create subsections: rename a marker to e.g. 3-A, then M creates 3-B, 3-C etc.
· On delete: remaining markers renumber from deletion point per cascade rules.
SEGMENT ASSEMBLY
Any marker can be a start or end point. A pool of 4 markers (0,1,2,3) can yield segments 0→1, 1→2, 2→3, 0→2, 1→3, 0→3 — all independent. "All adjacent" adds only consecutive pairs as a starting point.
KEYBOARD
SPACE play/pause · M mark · P play to next marker · Home back to start · Ctrl/⌘+Z undo · Esc close modal
Note: R/D marker type distinction has been removed. Single marker type only.
