REHEARSAL SPACE v5.1
WHAT'S NEW IN v5.1
· Folder architecture: Connect a Rehearsal Space folder on your Desktop. Each piece gets its own subfolder with session.json, notes.txt, and a recordings/ folder. Sessions save there automatically.
· Migration: On first connect, all localStorage sessions are written to the folder. A backup copy of the library JSON is written to the folder root on every export.
· Waveform tooltips: Hover over any marker line to see its label, bar number, and annotation.
· Consolidated recording: Single recording panel — clips appear inline with player, labels, format choice, and download. No separate review panel.
· Loop name on clips: Recorded clips are named after their loop (e.g. A→B) rather than a file timestamp.
· Stop after recording: Playback stops automatically after an armed loop recording completes.
· Folder permission must be re-granted each browser session — one click on the reconnect banner.
WORKFLOW
1. Select a piece from the Library — the file picker opens automatically.
2. Press Play. Press M to drop markers while listening.
3. In Loops: pick Start → End markers, click Add. Click ⟳ to loop, ▶ to preview.
4. Add notes in the Notes panel — saved automatically.
5. Save via Library → + Save current.
MARKER NAMING
· Numeric: 1 → 2 → 3
· Alpha: A → B → C
· Dash-numeric: A-2 → A-3 → A-4 (subsections)
· Dash-alpha: 3-A → 3-B → 3-C (subsections)
· Placeholder: . (period) — excluded from cascade and loop dropdowns
