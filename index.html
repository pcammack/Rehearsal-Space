MUSIC REHEARSAL v4.3
WORKFLOW
1. Select a piece from the Library, then load its audio file when prompted.
2. Press Play. Press M to drop markers while listening.
3. In Segments: pick Start → End, click Add. Or "All adjacent" for consecutive pairs.
4. Name segments, check/uncheck to include. Preview with ▶.
5. Save the piece via Library → + Save current.
MARKER NAMING
· Numeric: 1 → 2 → 3
· Alpha: A → B → C
· Dash-numeric: A-2 → A-3 → A-4
· Dash-alpha: 3-A → 3-B → 3-C
· Placeholder: . (period) — excluded from cascade and dropdowns
