MUSIC REHEARSAL v4.9
WHAT'S NEW IN v4.9
· Recording: Press R to start/stop recording. Clips appear immediately with playback and one-click download. Press K to discard the last clip.
· Bar numbers: Each marker now has an optional bar number field. Fill it in to navigate by musical position rather than memory.
· Recordings are session-only — download immediately if you want to keep them.
WORKFLOW
1. Select a piece from the Library — the file picker opens automatically.
2. Press Play. Press M to drop markers while listening.
3. In Loops: pick Start → End markers, click Add. Click ⟳ to loop, ▶ to preview.
4. Add notes in the Notes panel — saved automatically.
5. Save via Library → + Save current.
MARKER NAMING
· Numeric: 1 → 2 → 3
· Alpha: A → B → C
· Dash-numeric: A-2 → A-3 → A-4 (subsections)
· Dash-alpha: 3-A → 3-B → 3-C (subsections)
· Placeholder: . (period) — excluded from cascade and loop dropdowns
